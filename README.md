# Fibonacci Series Using Recursion in Java

## 📌 Description
This project demonstrates the Fibonacci Series using recursion in Java. The program accepts a number from the user, generates the Fibonacci sequence up to that count, and displays the Fibonacci value at the given position. It showcases the use of recursive functions, base cases, loops, and user input handling.

## ✨ Features
- Generates the Fibonacci series using recursion.
- Accepts user input with the Scanner class.
- Displays the Fibonacci sequence.
- Prints the Fibonacci value at the given position.
- Simple and beginner-friendly Java program.

## 🛠️ Technologies Used
- Java
- Scanner Class
- Recursion

## ▶️ Sample Input
Enter the number:
7

## 📤 Sample Output
0 1 1 2 3 5 8

Fibonacci value at position 7 is 13

## 📚 Concepts Covered
- Recursion
- Base Case
- Recursive Functions
- Loops
- User Input
- Method Calling

## ⚠️ Note
The recursive approach is easy to understand but is inefficient for large values because it repeatedly calculates the same Fibonacci numbers. An iterative or dynamic programming approach is recommended for better performance.
